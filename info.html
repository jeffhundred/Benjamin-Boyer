My name is Benjamin Boyer, and
welcome to my website! this site's 
sole purpose is to showcase my 
current abilities in web development.
While this site does contain a fair 
bit of redundancy, I believe it 
adequately diplays the base-level 
for my web design skills.
						
The majority of this website is
written in pure HTML and CSS, with
the exception of the couple of games
on the site, which were coded using
Javascript. I used the Sublime text
editor to code the entire site, and
the Javascript games implement the
P5.js library.
						
This website is designed to be
responsive, and has been optimized
for both mobile and desktop use.
while I have not created a version 
for tablet, by using media queries,
the site is made to be resposive to 
screen size (e.g. a laptop with a 
screen size above 1000 pixels)
